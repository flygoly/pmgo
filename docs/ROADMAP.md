# Roadmap

`pmgo` follows a phased rollout:

- **M1 (MVP):** repository scaffold, core skills, one PM integration, one IM channel
- **M2 (Beta):** risk radar, expanded integrations, active cron/heartbeat orchestration
- **M3 (v1.0):** broader connector coverage, richer visualizations, publishable skills pack

Milestone details are maintained in `README.md` and plan documents under `docs/plan/`.
