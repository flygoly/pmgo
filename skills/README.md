# Skills Pack Scaffold

The `skills/` directory hosts MCP-compatible capabilities for `pmgo`.

Planned modules include:

- `project-core`
- `daily-standup`
- `risk-radar`
- `weekly-report`
- `integration-github`
- `integration-jira`
- `integration-linear`
- `integration-notion`
- `integration-feishu`
- `integration-dingtalk`

Each skill should include its own `SKILL.md` and implementation files.
